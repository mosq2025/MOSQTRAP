SYSTEM COMPONENTS

  • Raspberry Pi 5
  • Camera Module
  • UV Light Attractor
  • Fan Trap Mechanism
  • Relay Module
  • 12V Power Supply
  • Web-based Dashboard

MAINTENANCE

  • Clean the trap weekly
  • Check camera lens for dust
  • Ensure UV light is functioning
  • Verify power connections

SYSTEM SETUP

  1. Connect the camera to the Raspberry Pi.
  2. Install the UV light and fan inside the trap.
  3. Connect the relay module to control the fan.
  4. Power the system using a 12V power supply.
  5. Ensure stable internet connection.

SYSTEM OPERATION

Once powered on, the system automatically activates the camera and UV light. Mosquitoes attracted by the UV light are captured by the fan trap and detected using the YOLO-based machine learning model.

SAFETY GUIDELINES

Always turn off the power before servicing the system. Avoid direct exposure to UV light and ensure proper insulation of electrical components. Keep the system away from moisture and ensure proper ventilation.
