MOSTRAP: MOSQUITO SMART TRAP FOR REAL-TIME ASSESSMENT AND PREVENTION OF AEDES AEGYPTI INFESTATION

SYSTEM OVERVIEW

MOSTRAP consists of the following Key Components:

  • Raspberry Pi (Main Processor)
  • Webcamera Module (Image Capture)
  • UV-LED Light (Mosquito attractant and Illumination)
  • Suction Fan (Capture Mechanism)
  • Collection Tray (Mosquito Chamber)
  • Web-based Dashboard (Monitoring Interface)

The system detects mosquitos using image processing and logs data for real-time monitoring and analysis

HARDWARE REQUIREMENT

  • Stable 12V power supply
  • Wi-Fi connection
  • MOSTRAP device (fully assembled)

SOFTWARE REQUIREMENT

  • Web browser (Chrome, Edge, etc.)
  • Internet connection for dashboard access

FEATURES

  • Real-time mosquito detection
  • Automated counting and logging
  • Early Warning Alerts (EWA)
  • Web-based monitoring dashboard
  • Data storage and trend analysis
  • Remote access via cloud database

INSTALLATION GUIDE

Device Placement

  • Place the device in areas prone to mosquito activity (e.g., near stagnant water, shaded outdoor areas).
  • Ensure the device is elevated or securely positioned.
  • Avoid direct exposure to heavy rain or extreme sunlight.

Power Setup

  1. Connect the device to a power source (12V supply).
  2. Turn ON the rocker switch.
  3. Wait for the system to initialize.

Wi-Fi Connection

  • The system automatically searches for available networks.
  • Ensure stable internet connection for real-time data syncing.

MAINTENANCE GUIDE

Cleaning

  • Empty the collection tray regularly
  • Clean mesh and fan area to prevent clogging

Inspection

  • Check wiring and power connections
  • Ensure camera lens is clean

Software Maintenance

  • Ensure stable internet for updates
  • Monitor dashboard for system alerts

OPERATING INSTRUCTIONS

Starting the System

  1. Turn ON the device.
  2. The system initializes all components (camera, sensors, fan, and Wi-Fi).
  3. The device enters monitoring mode during scheduled hours.

Monitoring Process

  • UV light attracts mosquitoes
  • Fan pulls mosquitoes into the trap
  • Camera captures images
  • System identifies if the insect is a mosquito
  • Data is logged and analyzed

Accessing the Dashboard

  1. Open the web application
  2. Log in using your credentials
  3. Navigate through:
    • Dashboard (real-time data)
    • Risk Assessment
    • History Logs
    • Notifications

SAFETY PRECAUTIONS

  • Do not expose the device to water directly
  • Use proper voltage supply only
  • Keep away from children during operation
  • Turn off before maintenance

LIMITATIONS

  • The system does not kill mosquitoes; it only monitors and captures
  • Performance may be affected by:
    • Weather conditions
    • Lighting variations
    • Internet connectivity issues
    • Power Outage
